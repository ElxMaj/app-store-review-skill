## App Feature Description

Nimbus Weather is a SwiftUI app built for iOS 16+. Key features:

- Current conditions (temperature, humidity, wind speed, UV index) for the user's current GPS location
- 7-day forecast with daily high/low and precipitation probability
- Hourly breakdown for the next 24 hours
- Severe weather alert banners sourced from the National Weather Service
- Dark mode and Dynamic Type support
- No third-party weather data SDK — uses Apple's WeatherKit framework directly
